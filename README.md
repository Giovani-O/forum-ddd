# forum-ddd

## Subdomínios

- Core: O que é lucrativo, tabalha com valores
- Supporting: Suporta o funcionamento do Core
- Generic: Necessários, mas não vitais

## Exemplos aplicados a um E-Commerce

### Core

- Compra
- Catálogo
- Pagamento
- Entrega
- Faturamento

### Supporting

- Estoque

### Generic

- Notificações ao cliente
- Promoções
- Chat
